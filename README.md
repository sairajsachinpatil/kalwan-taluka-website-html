# kalwan-taluka-website-html

🔗 **Live Website:** https://kalwaninfo.netlify.app/

---

## 📌 About the Project

The **Kalwan Taluka Website** is a responsive web project built using HTML and CSS.
It provides detailed information about Kalwan (Nashik district, Maharashtra), including villages, famous places, culture, and local lifestyle.

This project focuses on **clean design, simple navigation, and real-world website structure**.

---

## ✨ Features

* 🏠 Home page with introduction of Kalwan
* 🏘️ Villages page listing major villages
* 🏞️ Places page highlighting famous locations
* 🎭 Culture section
* 📩 Contact / Feedback form
* 🔗 Multi-page navigation system

---

* HTML5
* CSS3
